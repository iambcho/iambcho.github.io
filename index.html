React Pomodoro App

This is a React web app where you can specify a timer for your own Pomodoro
After you specify the amount of minutes, a timer will countdown on the screen until zero is reached.

MyBudget Web App

This is a Javascript web app where you can add, delete, swap (by dragging and dropping), and sum budget items.
This app makes use of local storage, regex (for error handling), and several different libraries.

Photo Filter App

This program is a photo filter app that asks the user for a file path to the image to be filtered.
The program can execute the following filters: grayscale, pixelate, kaleidoscope, flip (the image), and tetris.
The program can execute one or several filters on one image simultaneously.

Falling Blocks Game App

This is a game that utilizes the python turtle module to draw falling blocks and a user block that must avoid the falling blocks.
The user controls their block with the left and right arrow keys. The user must have the block avoid falling blocks or else the game will stop.
